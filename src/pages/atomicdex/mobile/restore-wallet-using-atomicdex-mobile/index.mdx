# How to Restore Wallet Using AtomicDEX Mobile

*   Start atomicDEX Mobile app

*   Click "RESTORE" button

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore1} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />

*   Give a memorable name to your wallet and press "LET'S GET SET UP!"

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore2} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />

*   Select "Allow custom seed" and paste or type your seed words

Make sure to include spaces in your seed, just as before.

If you would like to see what you are typing, click on the eye icon on the right to make the field visible.

*   Once ready, press the "CONFIRM" button

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore3} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />

*   Create a password to encrypt your seed

Make sure to use strong password with a combination of at least one capital letter, one small letter, a symbol, and numbers.

*   Press "CONFIRM PASSWORD"

If normal encryption takes longer on your phone, enable "Fast encryption" and try again.

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore4} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />

*   Create a PIN

In the next two steps you will be asked to provide a custom six digit PIN. You will need this PIN every time you want to access atomicDEX.

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore5} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />

<OptimizedImage title="How to Restore Wallet Using AtomicDEX Mobile" src={howToRestore6} classNaming="w-full" alt="How to Restore Wallet Using AtomicDEX Mobile" />
