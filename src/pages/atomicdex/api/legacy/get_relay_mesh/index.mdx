## get_relay_mesh

**get_relay_mesh**

The `get_relay_mesh` method returns a list of peerIDs included in our local relay mesh.

## Arguments

| Structure | Type | Description |
| --------- | ---- | ----------- |
| (none)    |      |             |

#### Response

| Structure | Type | Description                          |
| --------- | ---- | ------------------------------------ |
| PeerIDs   | List | PeerIDs within your local relay mesh |
