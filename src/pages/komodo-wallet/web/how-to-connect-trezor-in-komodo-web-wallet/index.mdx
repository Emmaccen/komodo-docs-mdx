# How to Connect Trezor in Komodo Web Wallet

*   In a new browser tab, open [Komodo Web Wallet](https://app.komodoplatform.com/#/dex)
    - *+Note:** Trezor is not supported on Safari and Firefox browsers

*   Click the “Connect wallet” button

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor1} alt=" How to Connect Trezor in Komodo Web Wallet" />

*   Click the “Connect Hardware wallet” button

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor2} alt=" How to Connect Trezor in Komodo Web Wallet" />

*   Select TREZOR and press the “Continue”

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor3} alt=" How to Connect Trezor in Komodo Web Wallet" />

*   Then, you will see the pop-up window with all connected Trezor devices.
    - **Note:** if you use Trezor Model T you need to enter a PIN code on the graphic screen to see the device in this list

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor4} alt=" How to Connect Trezor in Komodo Web Wallet" />

*   If you use Trezor Model One you need to enter a PIN with the Komodo Wallet interface after selecting wallet from the list.

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor5} alt=" How to Connect Trezor in Komodo Web Wallet" />

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor6} alt=" How to Connect Trezor in Komodo Web Wallet" />

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor7} alt=" How to Connect Trezor in Komodo Web Wallet" />

    - Then click the “Continue” button

*   Then, you will see the passphrase window for a hidden wallet. You can enter the passphrase and continue or skip this step to continue without a passphrase

<OptimizedImage title=" How to Connect Trezor in Komodo Web Wallet" src={howToConnectTrezor8} alt=" How to Connect Trezor in Komodo Web Wallet" />

*   Now, you have access to your trezor wallets :tada:
