# How to Do a Maker Swap in Komodo Web Wallet

*   In a new browser tab, open [Komodo Web Wallet](https://app.komodoplatform.com/#/dex) and [CREATE](src/pages/komodo-wallet/web/how-to-create-wallet-in-komodo-web-wallet/index.mdx) or [IMPORT](src/pages/komodo-wallet/web/how-to-import-wallet-in-komodo-web-wallet) your wallet

*   Open the DEX tab

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap1} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   Click the “Maker order” button at the top right of the box in the center of the screen.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap2} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   Click on the coin image in the “Sell coin” input at the top of the form to see the coin selection menu.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap3} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

    - At the top of the list, you will see your [activated]() coins with balances. Below in the list are all other coins available on the AtomicDEX Web which are not activated - if you select one of these, the coin will activate automatically!

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap4} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

    - Select the coin you want to sell by selecting it from list.

*   After selecting the sell coin you will see the available balance for this coin on above the inputs. The available balance will be slightly less than your full balance, to account for estimated transaction and DEX fees for a swap.
    - You can enter how many coins you want to sell by typing the amount into the sell coin input area, or by clicking the "50% or “Max” buttons to sell half or all of your tradable balance.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap5} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   Click on the coin image in the “Buy coin” input to select the coin you want to buy.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap6} alt=" How to Do a Maker Swap in Komodo Web Wallet" />


*   On the right of the form, an *+Order Book** will appear for you to review the all available price and volume options for this pair.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap7} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

    - In the **Maker order** form you can **create** any order you want by entering a buy amount for buy coin. You will see the price bellow.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap8} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   Click the “Make Order” button to see the confirmation page.

    - Please confirm all order details are correct before confirming the order.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap9} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   Now, this order has been added to the orders tab. All users can see you order and trade with you.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap10} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

    - Click on order to see more details about it.

<OptimizedImage title=" How to Do a Maker Swap in Komodo Web Wallet" src={howToDoMakerSwap11} alt=" How to Do a Maker Swap in Komodo Web Wallet" />

*   You can cancel on order before somebody starts to trade with by clicking “cancel” button
